# Cloud Computing Basics

A beginner project that demystifies cloud computing from the ground up. Understand the service models (IaaS, PaaS, SaaS), the major providers and the shared-responsibility model, virtualization and why it underpins everything, cloud networking (regions, VPCs, security groups), object vs block storage, and infrastructure as code with Terraform. You finish by provisioning a virtual private server with Terraform and deploying a real web app onto it — turning abstract cloud concepts into a running URL you understand end to end.

Built step-by-step with [KhwajaLabs Build](https://khwajalabs.com).

## Stack
- Cloud
- Linux
- Terraform
- Nginx
- Bash
