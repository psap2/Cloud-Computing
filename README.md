# Cloud Computing Basics

Project to understand the service models (IaaS, PaaS, SaaS), the major providers and the shared-responsibility model, virtualization and why it underpins everything, cloud networking (regions, VPCs, security groups), object vs block storage, and infrastructure as code with Terraform. Provisioned a virtual private server with Terraform and deployed a real web app onto it — turning abstract cloud concepts into a running URL.

## Stack
- Cloud
- Linux
- Terraform
- Nginx
- Bash
